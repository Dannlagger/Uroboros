# Uroboros

Sitio experimental local para practicar construcción web con transiciones ligadas al scroll.

## Vista local

Abre `index.html` directamente o sirve esta carpeta con cualquier servidor estático.

## Archivos principales

- `index.html`
- `styles.css`
- `script.js`

## Objetivo

Explorar un sitio visual y dinámico con imágenes que se fragmentan en partes durante el scroll.
